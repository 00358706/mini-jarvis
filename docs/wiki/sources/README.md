# Sources

This folder collects pointers to “source of truth” references used to justify wiki summaries.

Examples of entries:
- `source-files.md` — key codepaths and where invariants are enforced
- `tests.md` — smoke tests and what they guarantee
- `workspaces.md` — examples of workspace ids that captured important behavior

Guideline: prefer **links/pointers** over copying large raw content.
